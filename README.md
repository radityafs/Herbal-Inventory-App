# Herbal Inventory App

## Description
this is a simple app to track the inventory of herbs

## Features
- [x] Add herbs
- [x] Remove herbs
- [x] Edit herbs
- [x] Search herbs

## Installation
1. Clone the repo
2. Build the app
3. Run the app
4. Enjoy

## Libraries
* [Retrofit](https://square.github.io/retrofit/)
* [Gson]
* [Picasso] (https://square.github.io/picasso/)
